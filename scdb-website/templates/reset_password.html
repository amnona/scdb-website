Password recovery page
User/Password are used in order to create annotations that only you will be able to change (whereas anonymous annotations can edited by all users). Currently, non-anonymous users are supported using the Calour (https://github.com/biocore/calour) dbbact client.
To recover existing password, temporary code will be sent to the mail which was used during user's registeration.
In case there is no registered mail, please contact us via email info@dbbact.org
